# Multiple project SBT build

## SBT Commands

```bash
# start the sbt
sbt
```

```sh
reload
compile
run
test
```

```sh
# list all projects
projects

# display the versions of all projects
show version

# compile a particular project
util1 / compile

# run a particular project
util1 / run

# test a particular project
util1 / test
```
